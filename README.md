# 🚨 ResQHer
### IoT-Based Women's Safety Emergency Alert System using ESP32

ResQHer is an **IoT-based wearable emergency alert system** designed to enhance women's safety. The system allows users to trigger an SOS alert instantly using a **double-tap gesture on a capacitive touch sensor**, which sends emergency notifications along with location data.

---

## 📌 Abstract

Women's safety remains a critical concern in both urban and rural environments. In emergency situations, individuals often lack a quick and reliable way to send alerts and share their location with trusted contacts or authorities.

ResQHer is a wearable IoT device built using **ESP32** that enables users to trigger an SOS alert instantly. When activated, the system processes the signal, retrieves the user's location, and sends alerts to predefined contacts while storing data in the cloud for emergency response.

---

## ⚙️ System Architecture

1. **Wearable Device Layer**
   - Capacitive touch sensor detects user activation.

2. **ESP32 Processing Unit**
   - Processes the double-tap trigger.
   - Generates GPS/location data.

3. **Cloud Storage**
   - Emergency alerts and location data stored in Google Sheets or cloud database.

## 🔄 Working Principle

1. User **double-taps the touch sensor**.
2. The **ESP32 detects the activation**.
3. The system **generates location data**.
4. An **SOS alert is sent to predefined contacts**.
5. The **data is stored in the cloud** for emergency tracking.
